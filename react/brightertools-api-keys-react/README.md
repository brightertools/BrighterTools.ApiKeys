# @brightertools/api-keys-react

React companion components and localization helpers for `BrighterTools.ApiKeys`.

The host application owns authentication, localization storage, and HTTP response envelopes. This package owns reusable API key management UI orchestration and exposes an adapter-first `ApiKeysApiClient` contract.

## Validation

```text
npm install
npm test
npm run build
npm run pack:dry-run
```
